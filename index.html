object_recognition_linemod: Object Recognition Using LINE-MOD¶

This pipeline implements LINE-MOD. For more information, see http://ar.in.tum.de/Main/StefanHinterstoisser. It is one of the best methods out there for generic rigid object recognition and it proceeds using very fast template matching. The version used in that package is the same as the original paper (and it is in OpenCV) but the tricks to get it work properly are in the pre/post processing steps.
